The Development of CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language. CSS enables the separation content from presentation of the document, which makes the presentation more flexible and controllable. The CSS specifications are maintained by the World Wide Web Consortium (W3C). The W3C operates a free CSS validation service for CSS documents.

CSS1 (1996) is the first CSS specification to become an official W3C Recommendation. It supports font properties, element colour, text attributes, alignment, positioning, Unique identification and generic classification of groups of attributes.

CSS2 (1998) is a superset of CSS1, which includes more capabilities such as absolute, relative, and fixed positioning of elements and z-index, the concept of media types, support for aural style sheets, bidirectional text, and new font properties such as shadows.

CSS3 (1999-2012) is divided into several separate documents called â€œmodules". Each module adds new capabilities or extends features defined in CSS 2, preserving backward compatibility. The first draft was released in 1999, and later fifty modules were published, among which four have been included in the formal recommendations, which are Colour, Selectors Level 3, Namespaces(2011) and Media Queries(2012).
