1960s

ARPANET was established, being the first network employing packet switching technology. It is considered to be the ancestor of Internet.

Ted Nelson coined the terms “hypertext” and “hypermedia” to describe branching and responding text and multimedia.

The first public demo of ARPANET was released.
Vinton Cerf and Robert Kahn design TCP during 1973 and later publish it.
Robert Metcalfe creates the Ethernet.
Jobs and Woz take Apple public, launching the PC revolution.

1970s
1980s

Domain name server (DNS) is introduced.
Apple Macintosh ushers in “desktop publishing”, empowering citizens to become designers with mixed, but sometimes wonderful results.
Tim Bray and others co-found Open Text, the first internet search engine.

Tim Berners-Lee, invents the World Wide Web, routing the idea in hypertext.
Marc Andreeson and Eric Bina invent a point-and-click graphical browser for the web, called Mosaic.
Netscape introduces proprietary tags (IMG, EMBED) that are not part of HTML.
Tim Berners-Lee founds the World Wide Web Consortium (W3C), an international non-profit think tank dedicated to providing a rational roadmap for technological evolution of the web.

Jakob Nielsen begins publishing articles calling for a conservative, rules-based, user-focused approach to web development, which is called “usability”.
The rift between designers and usability experts will not heal until the mid 2000s.

Microsoft Internet Explorer 3.0 begins supporting CSS. It also supports a reverse engineered proprietary version of JavaScript.
HTML for Dummies is published.

Netscape 4.0, which supports advanced JavaScript, and supports some CSS which is more inaccurately than Internet Explorer 4.0.

1990s
2000s

Internet Explorer 5, Macintosh Edition, is the first mainstream browser with near-perfect support for HTML5, CSS1, and JavaScript. It also supports DOCTYPE switching, a mechanism that will ultimately rescue CSS from implosion.
Netscape 6 supports XML and the W3C DOM as well as the standards supported by IE5/Mac.
Opera 5 supports HTML, CSS, XML, WML, ECMAScript, and the W3C DOM.
Eric Meyer published CSS: The Definitive Guide, explaining in detail how CSS lets developers separate a web page’s structure from its presentation.

Taking Your Talent To the Web is published and describes an approach that completely overthrows the HTML table and spacer GIF hacks with which the web has been built.
Todd Fahrner publishes the first CSS method for setting font sizes without creating accessibility problems in Internet Explorer.

Microsoft releases Internet Explorer 6.0, which folds a great many standards fixes and upgrades into its DOCTYPE switching.
For the first time, developers who had held off implementing standards-based design because “IE doesn’t support it” taste the possibility of writing CSS that is truly cross-browser-compatible.

The CSS Working Group publishes the first draft of CSS2.1.
Position Is Everything, or PIE, begins documenting the existence of and fixes for numerous CSS bugs, browser quirks, and common layout problems.

Apple released Safari, a Webkit-based browser which will quickly become the most standards-compliant in the world.
Jeffrey Feldman writes Designing With Web Standards, which brings standards awareness to an international audience.

Jesse James Garrett coins the term “Ajax” to describe using JavaScript to post bits of data to a server, get bits back, and dynamically update pages.
CSS Mastery is published by Andy Budd, Simon Collison, and Cameron Moll; and Web Accessibility by a stellar collection of experts.

Steve Jobs introduces the iPhone. Millions of consumers begin experiencing rich web-standards-created experiences on their phones.
Advanced web standards compliance is baked into their new mobile experience.

The W3C quietly kills XHTML 2.0. HTML5 explodes into community consciousness.
HTML5 is a combination of new elements and attributes and the first major markup advancement in years.

Steve Jobs kills Flash in favour of HTML5. Adobe immediately pipes up to say its not dead yet.
CSS2.1 finally becomes a full W3C Recommendation.

The HTML5 programming language is recommended and released to the public in October, 2014 by W3C.
